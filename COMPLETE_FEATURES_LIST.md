## 🚀 How to Use

### For Users:
1. **Login/Register** - Create account or login
2. **Mark Attendance** - Click on attendance card → Mark Present
3. **View Plans** - Click membership card → Select plan → Pay
4. **Track Progress** - Profile → Progress Tracking → Enter measurements
5. **View Bills** - Profile → View Bills & Receipts → Download PDF

### For Admins:
1. **Manage Members** - Members tab → Add/Edit/Delete
2. **View Dashboard** - See statistics and overview
3. **Settings** - Drawer → Settings → Update gym details
